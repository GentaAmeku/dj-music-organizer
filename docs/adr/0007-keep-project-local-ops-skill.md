# ADR 0007: プロジェクトローカルの運用スキルを持つ

## Status

Accepted

## Context

DJ Music Organizer では、WSLとPowerShellの引用差、USBドライブのマウント、音源ファイルのdry-run/apply、レポート索引、非ASCIIファイル名の扱いなど、プロジェクト固有の運用知識が多い。

これらを会話履歴だけに依存すると、次回以降に同じミスが起きやすい。

## Decision

`skills/dj-music-organizer-ops` をプロジェクト内に作成し、運用上の注意と手順をスキル化する。

初期段階では `~/.codex/skills` へ自動導入せず、リポジトリ管理されたプロジェクトローカルスキルとして保管する。

必要になったら、後でユーザー管理のスキルディレクトリへコピーまたはインストールする。

## Consequences

プロジェクトの作業者は、リポジトリ内のスキルを読めば安全な操作手順を再現できる。

一方で、自動発見されるグローバルスキルではないため、使う際は明示的に参照するか、後でインストール作業が必要になる。
