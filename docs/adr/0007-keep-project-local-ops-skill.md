# ADR 0007: プロジェクトローカルの運用スキルを持つ

## Status

Accepted

## Context

DJ Music Organizer では、WSLとPowerShellの引用差、USBドライブのマウント、音源ファイルのdry-run/apply、レポート索引、非ASCIIファイル名の扱いなど、プロジェクト固有の運用知識が多い。

これらを会話履歴だけに依存すると、次回以降に同じミスが起きやすい。

## Decision

`.codex/skills/dj-music-organizer-ops` をプロジェクト内に作成し、運用上の注意と手順をスキル化する。

初期段階ではユーザー単位の `~/.codex/skills` へコピーせず、リポジトリ管理されたCodexローカルスキルとして保管する。

必要になったら、後でユーザー管理のスキルディレクトリへコピーまたはインストールする。

## Consequences

プロジェクトの作業者は、リポジトリ内のスキルを読めば安全な操作手順を再現できる。

一方で、Codexの実行面によってはリポジトリ内 `.codex/skills` が自動発見されない可能性がある。その場合は明示的に参照するか、ユーザー管理のスキルディレクトリへインストールする。
